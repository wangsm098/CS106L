# CS106L : Standard C++ Programming

CS106L is a class that explores the modern C++ language in depth. The course covers some of the most exciting features of C++, including modern patterns (up through C++26) that give it beauty and power. 

## Course Overview

[The course](https://web.stanford.edu/class/archive/cs/cs106l/cs106l.1262/) contains 16 lectures and 8 assignments.

### Lectures

- Welcome
- Types & Structs
- **Initialization & References**
- Streams
- Containers
- **Iterators & Pointers**
- Classes
- Inheritance
- **Class Templates & Const Correctness**
- **Function Templates**
- **Functions & Lambdas**
- Operator Overloading
- Special Member Functions
- **Move Semantics**
- std::optional & Type Safety
- **RAII, Smart Pointers, & Building C++ Projects**

The **bold** lectures are very comfortable and useful for me to learn and understand modern C++.

### Assignments

- Setup
- SimpleEnroll
- Marriage Pact
- Make a class!
- Ispell
- Treebook
- ExploreCourses
- Unique pointer

All the assignments will help you understand the knowledge you have learned and write better code.
